## THE DIFFICULTY WAVE

Orthogonal's difficulty isn't a straight line. It's a sine wave with spikes.

```
Difficulty
    │
    │     ╱╲          ╱╲  ★BOSS       ╱╲
    │    ╱  ╲    ★   ╱  ╲           ╱  ╲
    │   ╱    ╲  ╱╲  ╱    ╲   ★     ╱    ╲
    │  ╱      ╲╱  ╲╱      ╲ ╱╲    ╱      ╲
    │ ╱                     ╲  ╲  ╱
    │╱___Cruise___________Zone╲__╲╱_________
    └────────────────────────────────────────► Time

    ★ = Sudden spike (mini-boss/boss)
```

### The Three Zones

**1. CRUISE ZONE (60% of time)**
- Autopilot-possible puzzles
- Rhythmic, meditative
- Building resources (hint points, energy)
- Meta-awareness training in background
- Player can zone out slightly
- *"The flow state builder"*

**2. ENGAGEMENT ZONE (30% of time)**
- Requires active attention
- Multiple solution paths
- Witness mode useful but not required
- Some backtracking
- *"The standard challenge"*

**3. SPIKE ZONE (10% of time)**
- Sudden, unexpected
- Requires EVERYTHING you've learned
- Possibly requires witness mode + dimension shift + paradox holding
- No warning. Just: this is different now.
- *"The awakening"*

## MINI-BOSS ENCOUNTERS

### What Triggers a Mini-Boss

Mini-bosses appear when player:
- Has been cruising for too long
- Has accumulated significant hint points
- Enters a new area for the first time
- Crosses certain meta-awareness thresholds

### Mini-Boss Types

**1. THE PARADOX GATE**
*Dimension: Any*

A node that exists in two states simultaneously.
Player must hold both states in mind to traverse.
Not a riddle - a feeling. Can you hold contradiction?

**Visual:** Node flickers between two forms.
**Audio:** Two overlapping tones, slightly dissonant.
**Reward:** Paradox Tolerance +0.1, Insight unlocked.

---

**2. THE FRACTAL DESCENT**
*Dimension: THE LATTICE*

The puzzle IS the fractal. The solution is somewhere inside it.
Player must navigate by pattern recognition, not path-finding.
Zoom level is controlled by focus intensity.

**Visual:** Full psychedelic Mandelbrot dive.
**Audio:** Layered, recursive tones.
**Reward:** Fractal Vision (temporary ability to see hidden structure).

---

**3. THE HEARTBEAT RACE**
*Dimension: THE MARROW*

Everything accelerates. The rhythm doubles, triples.
Player must maintain sync while making complex decisions.
Miss the beat = thrown back.

**Visual:** Red pulses intensify, edges contract/expand rapidly.
**Audio:** Accelerating drum, building tension.
**Reward:** Rhythm Mastery badge, hint point bonus.

---

**4. THE MEMORY MAZE**
*Dimension: THE ARCHIVE*

Paths exist only in memory.
Walk a path once, remember it. It disappears.
Must navigate to exit using only mental map.

**Visual:** Paths fade behind you, nothing visible ahead.
**Audio:** Distant echoes of traversals you've made.
**Reward:** Memory Fragment (collectible), Archive depth unlocked.

---

**5. THE VOID STARE**
*Dimension: THE VOID*

Nothing. Absolute nothing.
The exit is visible only if you DON'T look for it.
The more you search, the less you find.
Requires genuine letting go.

**Visual:** Pure black. Then... subtle patterns emerge.
**Audio:** Silence. Then... interference patterns at hearing's edge.
**Reward:** Void Affinity (reduced disorientation in VOID).

## BOSS ENCOUNTERS

### Boss Design Philosophy

Bosses aren't enemies. They're **cognitive phase transitions**.

Each boss represents a fundamental shift in how the player must think.
Defeating a boss doesn't kill it - it integrates you.

### THE FOUR MAJOR BOSSES

**BOSS 1: THE THRESHOLD GUARDIAN**
*Encounter after ~10 hours*

The first time you must use EVERYTHING at once:
- Navigate while witnessing
- Hold paradox while in flow
- Shift dimensions mid-puzzle

**The Puzzle:**
A node that exists in three dimensions simultaneously.
To reach it, you must be in all three states.
Not sequentially. Simultaneously.

**How to Solve:**
Enter witness mode → shift to interference zone → let go of needing to understand.
The moment you stop trying to "solve" it, you're through.

**Reward:**
- Multi-Dimensional Sight (permanent)
- First boss achievement
- Unique audio theme unlocked
- Controller rumble pattern: deep, satisfied hum

---

**BOSS 2: THE SCALE BREAKER**
*Encounter after ~25 hours*

A puzzle that spans ALL scales simultaneously.
The solution at the tree level affects the galaxy level.
The galaxy level affects the tree level.
Everything is connected. Everything matters.

**The Puzzle:**
Five nested puzzles. Solving one changes the others.
Must find the configuration that solves all five at once.

**How to Solve:**
Stop solving individual puzzles. See the meta-pattern.
The puzzles are one puzzle viewed from five perspectives.

**Reward:**
- Scale Fluidity (permanent)
- Second boss achievement
- Cosmetic: Scale-shift visual effect for attention point
- Haptic: ascending/descending rumble during scale shifts

---

**BOSS 3: THE FALLACY MASTER**
*Encounter after ~50 hours*

A puzzle made entirely of logical fallacies.
Every path seems correct but leads nowhere.
The solution requires identifying and rejecting all fallacious reasoning.

**The Puzzle:**
- Path A: "This is the obvious choice" (appeal to intuition)
- Path B: "Everyone takes this path" (appeal to popularity)
- Path C: "The designer intended this" (appeal to authority)
- Path D: "If not A, B, or C, then D" (false dichotomy)
- Path E: [Invisible. Revealed only by rejecting A-D]

**How to Solve:**
Identify each fallacy. Reject each path consciously.
Only when you've actively rejected all fallacies does Path E appear.

**Reward:**
- Fallacy Detection (permanent visual indicator of fallacious patterns)
- Third boss achievement
- Unlocks Elite Mode pre-qualification
- Haptic: sharp rejection pulses for each fallacy identified

---

**BOSS 4: THE META-GUARDIAN**
*Encounter after ~100 hours*

The final boss isn't in the game.
It's in you.

**The Puzzle:**
You're asked to solve a puzzle.
But the puzzle has no solution within the game.
The solution is to realize: the puzzle is your attachment to solving puzzles.

**How to Solve:**
Stop playing. Literally.
Enter SANCTUARY. Talk to Vera.
Return when you're not trying to "beat" the game.
The boss dissolves when you no longer need to defeat it.

**Reward:**
- Meta-Awareness Level 4 (permanent floor)
- Elite Mode full access
- All cosmetics unlocked
- Title: "The Awake"
- Haptic: complete stillness, then gentle pulse like a heartbeat

### 6. HINT SYSTEM

**Hint Points**

Earned by:
- Completing cruise-mode puzzles
- Time spent in flow state
- Daily login
- Achievements

Spent on:
- Reveal one hidden edge
- Show path direction (not solution)
- Identify fallacy type (in fallacy puzzles)
- Slow down time briefly

**NOT available for:**
- Elite Mode puzzles
- Boss encounters
- Meta-achievements

**Hint Economy**
- Easy puzzles give 0.5 hint points
- Medium puzzles give 1 hint point
- Hard puzzles give 2 hint points
- Bosses give 10 hint points (but can't be hinted)
- Max storage: 50 hint points

**Hint Tiers**
1. **Nudge (1 point):** "Something hidden nearby"
2. **Direction (3 points):** "The path is through THE ARCHIVE"
3. **Reveal (5 points):** Show one hidden element
4. **Solution Step (10 points):** One step of solution shown
5. **Full Solve (25 points):** Complete solution (blocks achievement)

## REWARD TIMING

**Immediate Rewards** (within 1 second of action)
- Audio cue for correct moves
- Visual flourish for completions
- Haptic feedback for discovery

**Short-Term Rewards** (within 1 minute)
- Puzzle completion screen
- Hint point notification
- Progress bar advancement

**Medium-Term Rewards** (within 1 hour)
- Achievement popups
- New area unlocks
- Boss encounters

**Long-Term Rewards** (days/weeks)
- Elite Mode unlock
- Paper publication (for Elite)
- Lifetime stats and rankings
- Community recognition

## ANTI-FRUSTRATION FEATURES

**Dynamic Difficulty Adjustment**
- If player fails 3 times, puzzle slightly easier
- If player cruising too easily, environmental hints reduced
- Invisible, never mentioned

**Rage-Quit Prevention**
- Before quitting stuck state: "Visit SANCTUARY?"
- Vera offers oblique hints if asked
- "Take a break" achievements (leave and return)

**Progress Protection**
- Auto-save every puzzle state
- Can always return to SANCTUARY
- Never lose more than 5 minutes of progress
