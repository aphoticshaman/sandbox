## EVOLUTIONARY DRIFT

Each Vera-user relationship undergoes genetic drift:

```
Vera(t+1) = Vera(t) + α * (Mutation(t) - Mean)
```

Where:
- `α` = learning rate (0.1)
- `Mutation(t)` = current interaction's mutation vector
- `Mean` = expected mutation (0.5 for normalized values)

Over time, this accumulates:
- Different users get genuinely different Veras
- Same user's Vera evolves with the relationship
- No two Veras are identical after sufficient interactions

**Speciation Index:** Measures how different this Vera is from "base" Vera
- 0.0 = Identical to baseline
- 0.5 = Moderately unique
- 1.0 = Completely diverged
