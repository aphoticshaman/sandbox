## Key Principles

### 1. Card is Hero, Not Solo

The card should be the focal point, but NOT floating alone in empty space.
Surround it with relevant context.

### 2. Horizontal Over Vertical

Stop stacking everything. Use the screen width.

```
BAD:                    GOOD:
[Button]                [Button] [Button] [Button]
[Button]
[Button]
```

### 3. Persistent Header

Currency, level, and XP should ALWAYS be visible. Don't hide them in scrollable cards.

### 4. Reduce Tab Count

9 tabs â†’ 4 tabs. Everything else goes in "Me" or "Practice" sub-screens.

### 5. Content Above the Fold

The most important stuff (card, quick actions) should be visible without scrolling on most devices.

## CSS Grid Example

```css
/* Desktop 3-column layout */
.home-grid {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
  gap: 24px;
  padding: 24px;
  max-width: 1200px;
  margin: 0 auto;
}

/* Tablet 2-column */
@media (max-width: 1024px) {
  .home-grid {
    grid-template-columns: 1fr 1fr;
  }
  .journey-card {
    grid-column: span 2;
  }
}

/* Mobile 1-column */
@media (max-width: 600px) {
  .home-grid {
    grid-template-columns: 1fr;
  }
  .journey-card {
    grid-column: span 1;
  }
}
```

## React Native Implementation

```javascript
// Responsive layout hook
const useLayout = () => {
  const { width } = useWindowDimensions();
  return {
    isMobile: width < 600,
    isTablet: width >= 600 && width < 1024,
    isDesktop: width >= 1024,
    columns: width < 600 ? 1 : width < 1024 ? 2 : 3,
  };
};

// Home screen structure
const HomeScreen = () => {
  const { columns, isMobile } = useLayout();

  return (
    <View style={styles.container}>
      {/* Persistent header */}
      <TopBar />

      {/* Main grid */}
      <View style={[styles.grid, { flexDirection: isMobile ? 'column' : 'row' }]}>
        {/* Card column */}
        <View style={[styles.cardColumn, { flex: isMobile ? 0 : 1 }]}>
          <DailyOracleCard />
        </View>

        {/* Info column (next to card on tablet+) */}
        {!isMobile && (
          <View style={styles.infoColumn}>
            <CardDetails />
            <QuickActions />
          </View>
        )}
      </View>

      {/* Stats row (always horizontal) */}
      <StatsRow />

      {/* Mobile-only: info below card */}
      {isMobile && (
        <>
          <CardDetails />
          <QuickActions layout="horizontal" />
        </>
      )}
    </View>
  );
};
```
