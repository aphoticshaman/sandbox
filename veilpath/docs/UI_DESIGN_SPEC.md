### Component Details

**Featured Hero Card:**
- Large preview (300x450 on desktop, 200x300 tablet, 160x240 mobile)
- Animated glow based on rarity
- "NEW" or "SALE" badge
- Timer countdown if limited
- One-tap purchase with confirmation

**Category Tabs (Mobile):**
- Horizontal scrollable
- Active state: filled background + underline
- Badge for "new" items in category

**Item Grid Card:**
- Preview thumbnail (locked items show silhouette + lock icon)
- Name below image
- Rarity badge (color-coded)
- Price or "OWNED" / "EQUIPPED" status
- Tap â†’ detail modal OR instant equip if owned

**Item Detail Modal:**
- Full-size preview with 360 rotation on desktop
- Description + lore text
- Stats/effects (for animated items)
- [Purchase] / [Equip] / [Equipped] button
- Related items carousel

### Component Details

**Channel Sidebar:**
- Collapsible on mobile (hamburger menu)
- Active channel: highlight + indicator
- Unread badge (red dot + count)
- Premium channels: lock icon if not subscribed
- Channel description on hover (desktop)

**Post Card:**
- User avatar + name + time
- Content (markdown support)
- Embedded card preview (optional)
- Image attachment (optional)
- Like/Comment/Bookmark actions
- Reply count + preview of first reply

**Compose Area:**
- Expandable input
- Card picker (search/select)
- Image upload
- Post vs Chat toggle per channel
- Character count (500 max)

**Chat Mode (for realtime channels):**
- Bottom input bar
- Message bubbles
- Typing indicators
- Auto-scroll to new messages
- "Jump to present" button

**Trending Sidebar (Desktop):**
- Top shared cards today
- Top contributors (leaderboard)
- Badge progress hints
