## üìê ASSET SPECIFICATIONS SUMMARY

### Image Formats:
- **Icons:** PNG with transparency, 128x128px minimum
- **Backgrounds:** JPG or PNG, 1920x1080px (landscape), 1080x1920px (portrait)
- **UI Panels:** PNG with transparency
- **Particle Effects:** Sprite sheets or Lottie JSON

### Color Palette (Consistency):
- **Primary Purple:** #9945FF
- **Gold Accent:** #FFD700
- **Cyan Rare:** #00CED1
- **Deep Background:** #0A0A0A to #1A1A1A
- **White Highlights:** #FFFFFF

### File Size Targets:
- **Icons:** <50KB each
- **Backgrounds:** <500KB each
- **UI Panels:** <200KB each
- **Animations:** <300KB (Lottie) or <2MB (sprite sheets)
