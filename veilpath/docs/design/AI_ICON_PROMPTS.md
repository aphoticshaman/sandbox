## Quick Start Guide

### Midjourney (Recommended for Icon Design)

**How to Use:**
1. Join Midjourney Discord: https://discord.gg/midjourney
2. Subscribe to plan ($10/month minimum)
3. Go to any #general channel or DM the Midjourney Bot
4. Type `/imagine prompt: [paste prompt below]`
5. Wait ~60 seconds for 4 variations
6. Click `U1`, `U2`, `U3`, or `U4` to upscale your favorite
7. Download 1024x1024px PNG

**Best Practices:**
- Use `--v 6` or `--v 6.1` for latest model (better quality)
- Use `--style raw` for less artistic interpretation (more literal)
- Use `--ar 1:1` to ensure square output (app icons are square)
- Use `--no text` to avoid unwanted text/logos

---

### DALL-E 3 (via ChatGPT Plus)

**How to Use:**
1. Subscribe to ChatGPT Plus ($20/month)
2. Select "DALL-E 3" from model dropdown
3. Paste prompt and click "Generate"
4. Download image (1024x1024px)

**Best Practices:**
- Be specific about "app icon for mobile phone"
- DALL-E often adds text unless you specify "no text, no words"
- Request "simple, minimalist" to avoid over-detailing

---

### Stable Diffusion (Free, Advanced Users)

**How to Use:**
1. Install Stable Diffusion locally or use web UIs (e.g., Replicate, Leonardo.ai)
2. Use prompts below with adjustments for your model
3. Recommended settings: Steps 50-80, CFG Scale 7-10, Sampler: DPM++ 2M Karras

## General Parameters & Tips

### Midjourney Parameters Explained

**Model Versions:**
- `--v 6.1` (latest, best quality, most accurate)
- `--v 6` (very good, slightly more creative)
- `--v 5.2` (older, more artistic/abstract)

**Style:**
- `--style raw` (literal interpretation, less artistic flourishes) — **RECOMMENDED for icons**
- `--style original` (more creative/artistic)

**Aspect Ratio:**
- `--ar 1:1` (square, required for app icons)

**Quality/Detail:**
- `--q 2` (max quality, slower, more expensive credits)
- `--q 1` (default, good balance)

**Negative Prompts:**
- `--no text` (avoid text/words/letters)
- `--no watermark` (avoid logos)
- `--no people` (avoid realistic humans if using silhouettes)
- `--no complex details` (keep it simple)

**Chaos (Variability):**
- `--c 0` (very consistent, predictable) — **RECOMMENDED for icons**
- `--c 50` (default)
- `--c 100` (very chaotic, unpredictable)

---

### Universal Negative Prompts (What to Avoid)

Add these to any prompt to prevent common issues:

```
--no text, no words, no letters, no realistic people, no photographs, no 3D render, no complex patterns, no watermark, no signature, no frame, no border
```

**For Stable Diffusion (negative prompt field):**
```
text, words, letters, typography, realistic people, photograph, 3D, render, complex details, watermark, signature, frame, border, low quality, blurry, distorted
```

---

### Color Specification Tips

**For AI models, color descriptions work better than hex codes:**
- ❌ "Use #6B46C1" → AI may ignore exact hex
- ✅ "deep royal purple" → AI understands better
- ✅ "luminous teal, like turquoise gemstone" → More descriptive

**Best approach:**
1. Generate with descriptive colors
2. Adjust exact hex codes in Figma/Photoshop after generation

---

### Prompt Structure (Best Practices)

**Formula:**
```
[format] + [main subject] + [key details] + [colors] + [style/aesthetic] + [technical specs] + [negative prompts]
```

**Example:**
```
app icon design + winding path to moon + glowing teal path, gold moon, purple background + minimalist spiritual wellness + 1024x1024, square, flat vector + no text
```

## Post-Generation Editing

### Step 1: Cleanup in Photoshop/Figma

**If AI added unwanted elements:**
1. Use Magic Eraser or Object Selection to remove
2. Use Content-Aware Fill (Photoshop) to fill gaps
3. Redraw elements with vector tools (Figma/Illustrator)

**If colors aren't exact:**
1. Use Color Replace tool (Photoshop)
2. Select by color range → Hue/Saturation adjustment
3. Or redraw in Figma with exact hex codes

---

### Step 2: Vectorize (Optional)

**Why vectorize?**
- Scales to any size without quality loss
- Easier to adjust colors, shapes, spacing

**Tools:**
1. **Adobe Illustrator**: Image Trace (best quality)
2. **Figma**: Use Vector Networks plugin
3. **Vectorizer.ai**: Online tool (free)

**Steps (Illustrator):**
1. File → Place → [your PNG]
2. Select image → Image Trace → High Fidelity Photo
3. Expand → Ungroup
4. Clean up stray points
5. Adjust colors to exact hex codes

---

### Step 3: Test at Small Sizes

**Critical:**
1. Resize to 60x60px (iOS home screen size)
2. Check if icon is still clear and recognizable
3. If details are lost, simplify in Figma

**Common issues:**
- Thin lines disappear → Make thicker
- Small elements blur → Enlarge or remove
- Colors blend together → Increase contrast

---

### Step 4: Export All Required Sizes

**Use icon generator tools:**
- **Appicon.co**: Upload 1024x1024, auto-generates all iOS/Android sizes
- **Icon Slate** (Mac app): Batch export with naming
- **Android Asset Studio**: Generate adaptive icons

## Recommended Workflow

### Option A: Pure AI + Manual Cleanup (Fast)
1. Generate 10-20 variations with Midjourney
2. Choose best 3
3. Upscale to 1024x1024
4. Edit in Photoshop/Figma to fix colors, remove artifacts
5. Export all sizes
6. **Time:** 2-4 hours

---

### Option B: AI Inspiration → Full Redraw (Best Quality)
1. Generate 5-10 concepts with Midjourney
2. Choose favorite as reference
3. Fully redraw in Figma/Illustrator with vectors
4. Use exact brand colors, perfect spacing
5. Export all sizes
6. **Time:** 4-8 hours

2. Vectorize in Illustrator
3. Adjust colors to exact hex codes
4. Refine shapes, spacing, details in Figma
5. Export all sizes
6. **Time:** 3-6 hours

## Troubleshooting Common Issues

### Issue: AI keeps adding text/words
**Solution:**
- Add `--no text, no words, no letters` to Midjourney
- For DALL-E, explicitly state "absolutely no text, no words, no letters anywhere"
- Try `--style raw` in Midjourney for more literal interpretation

---

### Issue: Icon too detailed/complex
**Solution:**
- Add "minimalist, simple, clean, geometric" to prompt
- Use `--style raw` instead of default
- Request "flat design, vector illustration" style
- Manually simplify in post-processing

---

### Issue: Colors aren't right
**Solution:**
- Use descriptive color names ("deep purple, luminous teal, soft gold")
- Accept that AI won't match exact hex codes
- Plan to adjust colors in Figma/Photoshop after generation
- Or fully vectorize and recolor manually

---

### Issue: Icon looks too "AI-generated" (artifacts, weird details)
**Solution:**
- Use `--v 6.1 --style raw` for cleaner output
- Increase `--q 2` for higher quality
- Use `--c 0` for less chaos/variability
- Redraw in vector software using AI output as reference
